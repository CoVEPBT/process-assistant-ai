#25: Auto-apply by default + consent at home-page Analyse
======================================================

After Analyse, the AI-content consent is now captured once at
Analyse-click time, and auto-apply runs automatically on
completion.

- _doAnalyse() extracted from the click handler and exposed on
  window so the modal Proceed handler can call it.
- analyseBtn click gates on sessionAiConsent. First click opens
  the modal with _runAnalyseAfterConsent = true; later clicks
  in the same session bypass the modal.
- Modal Proceed branch: sessionAiConsent = true, run _doAnalyse,
  post-analyse hook auto-fires window.runAutoApplyDirect().
- Modal Cancel branch (relabelled "Review original first"):
  still runs _doAnalyse but skips auto-apply, so the user
  can review themselves. Manual auto-apply later re-shows the
  modal.
